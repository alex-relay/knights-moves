# Knights Moves

This project provides a solution to the chess knight movement challenge. It takes a given coordinate on an 8x8 chessboard and calculates all possible valid moves for a Knight piece.

## Overview
The program parses a standard chess coordinate (e.g., "E4"), validates the input, and returns a list of all potential moves based on the L-shaped movement rules of a Knight.

## Getting Started

### Prerequisites
- Python 3.x

### Usage
1. Ensure you have the `chess.py` file in your directory.
2. Run the script from your terminal: `python chess.py`
